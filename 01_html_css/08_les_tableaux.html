rowspan / colspan

rowspan est un attribut qui permet d'indiquer le nombre de ligne qu'une céllule <tb> de tablau <table peut couvrir. cet attribut est utile si voussouhaitez qu'une cellule s'tende verticalement sur plusieur ligne. Exemple si onnet rowspan="5" alors notre cellule occupera l'espace de 5 cellules verticalemen Cependant il est important de noter que l'orsqu' on utilise cette manipulation il faut retirer les tb correspondant des autre ligne

"colspan" est un attribut qui permet d'indiquer le nombre de colonnes qu’une cellule <td> de tableau <table> doit couvrir. Cet attribut est utile lorsque vous souhaitez qu’une cellule s’étende horizontalement sur plusieurs colonnes .Exemple si on met colspan="3", alors notre cellule occupera l'espace de 3 cellules horizontalement.
