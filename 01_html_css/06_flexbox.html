C - Direction

Flexbox nous permet d'agencer ces éléments dans le sen que l'on veut. Avec flex-direction, on peut les positionner verticalement ou encore les inverser. il peut prendre les valeurs suivantes:
-> row : organisés sur une ligne (par défaut)
-> column : organisés sur une colonne
-> row-reverse : organsés sur une ligne mais en ordre inversé
-> column-reverse : organisé sur une colonne, mais en ordre inversé

D- Alignement

Les éléments sont organisés soit horizontalement (par défaut), soit verticalement.

  • si vos éléments sont organisés horizontalement : l'axe principale (main axis) est l'axe horizontale, l'axe secondaire (cross axis) est l'axe vertical.
  • si vos éléments sont organisés verticalement : l'axe principale (main axis) est l'axe verticale, l'axe secondaire (cross axis) est l'axe horizontale.

1- Alignement sur l'axe principale

afin d'aligner nos élément sur l'axe principale : on utilise la propriété du flew justify-content

Cette propriété gère la distribution des emplacement des flex-items suivant la place disponible dans le flex-container. Elle le fait, selon le positionnement de l'axe principale

  • flex-star : aligner au début ( par défaut )
  • flex-end : aligner à la fin
  • center : aligner au centre
  • space-between : les éléments sont étirés sur tout l'axe (il y a de l'espace entre eux)
  • space-around : les élément sont étirés sur tout l'axe, mais ils laissent aussi de l'espace sur les extrémités
  • space-evenly : les éléments sont répartis uniformément dans le conteneur d'alignement le long de l'axe principal
